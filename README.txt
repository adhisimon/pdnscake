PDNS-Cake
--------
PDNS-Cake is a free and opensource web user interface to administer PDNS a.k.a PowerDNS (http://www.powerdns.com/).

Writen using CakePHP (http://www.cakephp.org) framework and support MySQL as database backend.

Development of this software has been initiated by PT. Mondial Teknologi Solusi (MTS) (http://www.mondial.co.id) and released under GPLv2 license.
